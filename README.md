# HoshiAI
An AI tool for real-time screen language translation by overlay text using Windows API

# Description
Hoshi AI is an advanced AI-powered application designed to provide real-time screen translation for users. It uses LLM (Large Language Model) technology to accurately translate text displayed on the screen into multiple languages, enhancing accessibility and user experience.

# Features
- Real-time screen text detection and translation
- Support for multiple languages
- User-friendly interface
- Customizable translation settings
- Detection of text from screen directly without the need for screenshots
- using Windows API for text overlays and capturing screen content
- OCR integration for text recognition
